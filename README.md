I build **end-to-end intelligent systems** — combining production-grade **ERP customization** with cutting-edge **AI/ML**. From fine-tuning LLMs and crafting **RAG pipelines** to developing cross-platform **Flutter** apps and deploying interactive demos on Hugging Face Spaces, I turn complex ideas into live, production-ready solutions.
